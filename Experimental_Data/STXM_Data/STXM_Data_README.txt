This folder contains all of the STXM data presented in the main text and extended data figures. Each folder contains sub folders with the raw data files, and .TIF files extracted from the raw images files. In each case, a .txt file details which the parameters of each raw file. A header (.hdr) file also contains the instrument conditions of each image. Some datasets also include Line_Scan folders containing m_z cuts through the images, performed with imageJ.
